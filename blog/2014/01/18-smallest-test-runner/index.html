Smallest test runner

Wolfram Kriesing - January 18, 2014 - tagged with: #tdd #testing

IDEs are awesome, but sometimes in my way and setting up test runners sometimes defeats the purpose of being fast with tests, which also means having feedback constantly and continuously.

Thanks to @Christoph for showing me the shortest test runner there is, it runs on the command line, like this:

while true; do npm test; sleep 3; done;

it assumes of course, your tests can be run via npm test.
